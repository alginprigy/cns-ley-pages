# cns-ley-pages

## 项目简介

本项目是一个用于归档和发布多个独立 HTML 页面的通用仓库。所收录的页面内容涵盖不同主题，旨在提供一个集中存放与展示静态 HTML 资源的空间。

本仓库不针对任何特定网站或域名，仅作为页面文件的托管与版本管理用途。

## 目录结构

```
cns-ley-pages/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放所有独立 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：核心目录，存放所有归档的 HTML 页面。
- `assets/`：页面可能依赖的公共样式、脚本或图片资源。
- `index.html`：可选的仓库首页，用于导航或展示页面列表。

## 页面归档说明

- 所有页面均以独立 HTML 文件形式存放于 `pages/` 目录下。
- 文件名应简洁清晰，建议避免特殊字符或空格。
- 每个页面应尽可能自包含，如需外部资源请统一放置于 `assets/` 中。
- 不强制要求页面之间有关联，允许独立存在。

## 维护说明

- 欢迎通过 Pull Request 提交新的页面归档，或对现有页面进行修正。
- 提交前请确保页面内容符合基本规范，无恶意代码或非法内容。
- 建议在提交时附带简要说明（如页面用途、修改内容等）。
- 本仓库不提供自动构建或部署脚本，页面以原始 HTML 形式存储。
